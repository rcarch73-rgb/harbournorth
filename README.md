# Harbour North 2.0 Alpha 2 Fixed

This build is intentionally self-contained. The full interface, styling, and application logic are embedded in `index.html` so GitHub Pages cannot load the page without its JavaScript.

## Install
1. Delete the old `index.html`, `app.js`, and `styles.css` from the repository.
2. Upload this `index.html` to the repository root.
3. Commit the change.
4. Reload the GitHub Pages site with a cache refresh.
